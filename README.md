## 📱 快速开始

### 1. 注册账号
1. 点击右上角「登录」按钮
2. 切换到「注册」标签
3. 填写信息：
   - **用户名**：唯一标识，注册后不可更改
   - **昵称**：显示名称
   - **入学年份**：2023/2024/2025
   - **班级**：1-25班
   - **密码**：至少6位，注册后不可更改
4. 上传头像（可选）
5. 点击「下一步」，确认信息无误后点击「确认注册」

⚠️ **重要提醒**：年级、班级、用户名、密码注册后均不可修改，请仔细核对！

### 2. 登录账号
- 输入用户名和密码
- 点击「登录」

### 3. 浏览帖子
- 首页展示所有帖子
- 使用顶部搜索框搜索关键词
- 左侧边栏可按标签、可见范围筛选

### 4. 发布帖子
1. 点击右上角「+ 发布」按钮
2. 填写标题和内容
3. 选择标签（伙食、八卦、老师、笔记等）
4. 设置可见范围：
   - 🏫 **全校**：所有人可见
   - 📚 **年级**：同年级可见
   - 👥 **班级**：同班同学可见
5. 添加图片（可选）：从设备相册选择，最多4张，自动压缩
6. 点击「发布」

### 5. 个人主页
- 点击右上角头像 → 「个人主页」
- 查看自己的发帖、获赞数
- 点击「编辑资料」修改：
  - 头像（点击头像区域上传）
  - 个人简介
  - 性别、生日、所在地
  - 爱好
  - 联系方式（手机号、微信、邮箱）

### 6. 查看他人主页
- 点击任意帖子中的用户头像
- 可查看该用户的公开信息和帖子
- 点击帖子可查看详情和评论

### 7. 互动功能
- **点赞**：点击心形图标
- **评论**：进入帖子详情，在底部输入评论
- **删除自己的帖子**：点击帖子右上角的删除按钮
